B.E.L.M.U (Base, Engine, Layout, Module, Utility) is a an organizational method for writing CSS, together with some layout helpers for grid and flexbox & a collection of utility classes. Think tachyons CSS with a faster learning curve (simply because there's not as much to learn)

Base

Base contains a normalize in the form of fix (https://github.com/jaydenseric/Fix). Base typography, forms and variables declared here.

Engine

The ideal is to have no Engine and just go completely vanila. But when you need things like svg icons, or other useful mixins you can define them here and build with postcss.
